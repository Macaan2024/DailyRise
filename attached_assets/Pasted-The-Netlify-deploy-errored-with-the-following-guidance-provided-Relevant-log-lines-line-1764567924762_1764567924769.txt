   - OR add engines to package.json:
```json

  }
}
```
   - OR set Netlify build environment in netlify.toml:
```toml
[build.environment]
  NODE_VERSION = "18.20.0"
```
   Commit and push whichever file you add.

2. Rebuild locally to verify:
   - Install/use the pinned Node: nvm install 18 && nvm use 18
   - Reinstall deps and run build: npm ci && npm run build
   - Fix any build errors that appear locally before redeploying.

3. Redeploy on Netlify. Netlify will pick up .nvmrc or package.json engines (or the NODE_VERSION env var) and use that Node version for the build.
